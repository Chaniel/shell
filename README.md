shell
=====
Shell Scripts Library.

### Powerful Shell Commands
- grep
- sed
- awk
- tr
  echo file | tr -s '[:space:]' ':' 
- cut
  cut -d":" -f1,3
- sort
  sort -k1 -nr
- uniq
  uniq -c
- ip
  ip addr, ip route, ...
- ss
  ss -s , ss -ntlp , ss -antp
- netstat 
`netstat -antp , netstat -ntlp , netstat -nutp`
